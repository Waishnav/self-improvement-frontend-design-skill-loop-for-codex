# T4 Canvas Marketing Homepage Experiment

A Vite + React + TypeScript + Tailwind project (Bun runtime) showcasing five distinct marketing homepage concepts for **T4 Canvas**.

## Stack

- Vite
- React 19
- TypeScript
- Tailwind CSS (via `@tailwindcss/vite`)
- Bun

## Routes

- `/1` Pulse Lab (hybrid canvas / kinetic feel)
- `/2` Editorial (print-inspired manifesto flow)
- `/3` Cockpit (dense command-center UI)
- `/4` Gallery (media-first compare surface)
- `/5` Poster (neo-brutalist poster web)

## Local Run

```bash
cd t4-canvas
bun install
bun run dev
```

Dev server is configured for port **4000**.

## What Changed in This Version

- Initialized a fresh Bun + Vite + React + TypeScript app in `./t4-canvas`.
- Integrated Tailwind into Vite.
- Added React Router routes for five standalone homepage directions.
- Implemented five fully different art-direction systems (layout, typography, color, and interaction style).
